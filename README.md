## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
# or
yarn install
```

### Development

```bash
# Start development server
npm run dev
# or
yarn dev
```

### Production Build

```bash
# Create production build
npm run build
# or
yarn build

# Serve production build
npm run start
# or
yarn start
```
